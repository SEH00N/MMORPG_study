## DB?
- 단순히 효율적인 데이터 관리를 위한 프로그램
- 데이터를 파일로 저장하고 불러온다면
  - 속도 문제
  - 멀티스레드 환경에서 동접 X
  - 안전 X
  - 데이터 분석 및 유지보수 등이 힘듦

## DB 종류
- RDBMS
  - Relational DataBase Management System
  - 관계형 데이터 베이스
  - 게임에 주로 사용되는 데이터 베이스
- NOSQL
  - Not Only SQL
  - 비관계형 데이터 베이스
  - 빅데이터 같은 경우에 유용함

## SQL?
